Demo 1C: Full, Static Grid

This demo uses just plain, static, fully rendered HTML, without using React or virtualization.

To see the larger initial display time and memory use:

  • Resize the grid with more rows. Try 5,000, 50,000 or 125,000 rows.
  • Watch for the message that reports how long it took to build the grid.
  • Optionally, open a utility that lets you also monitor memory usage.
